## 🚀 Cómo correr el proyecto

### Requisitos previos
- Node.js instalado ([descargar aquí](https://nodejs.org))
- pnpm instalado (ver paso 1)

### Paso 1 — Instalar pnpm
```
npm install -g pnpm
```

### Paso 2 — Entrar a la carpeta
```
cd studentconnect
```

### Paso 3 — Instalar dependencias
```
pnpm install
```

### Paso 5 — Aprobar scripts de Firebase
```
pnpm approve-builds (si dice que no hay para aprobar entonces no es necesario)
```
Presiona `a` para seleccionar todos → `Enter` → escribe `true` → `Enter`

### Paso 6 — Correr el proyecto
```
pnpm dev
```
